## FAQ - Frequently Asked Questions

### Submission Questions

**Q1: What documents do I need to submit?**
A: You must submit three documents:
1. Disclosure Form - Describes your IP in detail
2. Technical Drawings/Specifications - Diagrams and technical details
3. Supporting Documentation - References, market analysis, etc.

Download templates from: Settings > Templates

**Q2: What file formats are accepted?**
A: Accepted formats:
- PDF (recommended)
- DOCX (Word documents)
- XLSX (Excel spreadsheets)
- PNG, JPG (Images)
- ZIP (for multiple files)

Not allowed: EXE, COM, BAT, or any executable files

**Q3: What's the maximum file size?**
A: 
- Individual file: 10 MB maximum
- Total submission: 50 MB maximum
- Recommended: Keep files under 5 MB for faster upload

**Q4: How long does the review process take?**
A: Typical timeline:
- Received & Accepted: 2-3 days
- Under Review: 5-10 business days
- Awaiting Decision: 2-3 business days
- Certificate Issued: 1-2 days
- **Total: 10-18 business days**

**Q5: Can I upload documents after submission?**
A: Yes, you can:
1. Go to your submission
2. Click "Upload Additional Documents"
3. Upload new files
4. Supervisor will see updated submission

**Q6: What if I made a mistake in my submission?**
A: Before approval:
- Contact your supervisor to request revision
- Update incorrect information
- Resubmit updated documents
- Review restarts with updated info

**Q7: Can I delete a submission?**
A: Only before review starts:
1. Go to submission
2. Click "More Options" > "Delete"
3. Confirm deletion
4. Submission permanently removed

After review starts, contact system administrator.

**Q8: Can multiple inventors submit together?**
A: No, but you can add co-inventors:
1. Only one person submits
2. Add all inventors during "Inventors Information" step
3. All receive notification when reviewed
4. Certificate lists all inventors

**Q9: Do I keep my documents confidential?**
A: Yes:
- Only you, your supervisor, and assigned evaluators see documents
- System uses encrypted storage
- No public access
- GDPR and institutional privacy rules apply

**Q10: What if my submission is rejected?**
A: You can:
1. Request feedback on why rejected
2. Revise and resubmit
3. Contact supervisor for guidance
4. You have unlimited resubmission attempts

### Review & Evaluation Questions

**Q16: Who are the evaluators?**
A: Evaluators are:
- Subject matter experts in your field
- Faculty or specialists appointed by UCC
- Have no conflict of interest
- Bound by confidentiality

You don't choose evaluators; supervisor assigns them.

**Q17: What criteria do evaluators use?**
A: Evaluators assess:
- Innovation: Is it novel?
- Feasibility: Can it be built/implemented?
- Market Potential: Commercial viability?
- Technical Merit: Technical soundness?
- Each scored 0-10

**Q18: Can I communicate with evaluators?**
A: No direct communication, but:
- You can see their feedback/comments
- Any questions go through supervisor
- Supervisor acts as intermediary
- Ensures professional process

**Q19: What if I disagree with the evaluation?**
A: 
1. Contact your supervisor first
2. Request reconsideration
3. Supervisor can request re-evaluation
4. Final decision made by supervisor
5. Appeal process available - see system guide

**Q20: How long do evaluators have to complete review?**
A: 
- Typical: 5-10 business days
- If delayed: Supervisor sends reminders
- Status updated in your dashboard
- You can ask for status update from supervisor

### Certificate & Results Questions

**Q21: When do I receive my certificate?**
A: You receive certificate when:
- Your IP is approved
- Final paperwork completed (1-2 days)
- Certificate email sent
- Available for download

Check your email for "Certificate Ready" message.

**Q22: What does the certificate include?**
A: Certificate includes:
- Your name and all inventor names
- IP title
- Date issued
- UCC seal/approval
- Reference number
- Digital signature

**Q23: Can I print/share my certificate?**
A: Yes:
1. Download certificate from dashboard
2. Print on standard paper
3. Share with employers, partners
4. Use in presentations
5. Digital version valid and shareable

**Q24: What if I lose my certificate?**
A: You can always:
1. Log into your account
2. Go to submission
3. Click "Download Certificate"
4. Get fresh copy anytime
5. Never expires

**Q25: What happens after my IP is approved?**
A: You can:
- Patent it (file with patent office)
- License it (sell usage rights)
- Commercialize it
- Collaborate on development
- Share findings with others

UCC helps facilitate connections with industry partners.

### Technical Issues

**Q26: I see an error message - what should I do?**
A: 
1. Read error message carefully
2. Try the suggested solution
3. If not resolved: Refresh page (Ctrl+F5)
4. If still failing: Try different browser
5. If persists: Contact support with error text

**Q27: My file upload keeps failing**
A:
1. Check file size (max 10MB)
2. Check file format (PDF, DOCX, etc.)
3. Check file is not corrupted
4. Try different file
5. Try different browser
6. Check internet connection
7. Contact support if continues

**Q28: The system is very slow**
A:
1. Check internet connection
2. Close other browser tabs
3. Clear browser cache (Ctrl+Shift+Delete)
4. Try different browser
5. Try at different time
6. Contact IT if site consistently slow

**Q29: I keep getting logged out**
A:
- Normal if inactive for 30 minutes (security)
- Simply log back in
- If happens within minutes:
  1. Clear browser cache
  2. Check cookies enabled
  3. Try incognito mode
  4. Contact support

**Q30: The website doesn't look right on mobile**
A:
- System optimized for desktop viewing
- Mobile access limited but functional
- Use desktop for full features:
  - Document upload
  - Detailed review
  - Certificate download
- View-only features work on mobile:
  - Check status
  - Read messages
  - Download certificate

```
Different types of intellectual property need different approaches 
to protection and registration.

SOFTWARE & ALGORITHMS
- Computer programs and applications
- Code libraries and frameworks

- Machine learning models

HARDWARE & DEVICES
- Physical inventions and machines
- Electronic devices and components
- Medical devices
- Industrial equipment
Example: New type of microscope

PROCESSES & METHODS
- Manufacturing processes
- Business methods
- Scientific procedures
- Diagnostic methods
Example: Water purification process

PUBLICATIONS & CREATIVE WORKS
- Research papers
- Technical manuals
- Course materials
- Documentation
Example: Open-source documentation

DESIGNS
- Product designs
- User interface designs
- Industrial designs
- Graphic designs
Example: Novel container design

Why categorize?
- Different legal protections apply
- Different evaluation criteria
- Different commercialization paths
- Different inventor rights
```

### Article 2: "Preparing Your Disclosure Form"

```
Your disclosure form is the foundation of your IP application. 
It should clearly explain:

1. BACKGROUND
   - What problem does your innovation solve?
   - What is the current state of the field?
   - What existing solutions exist?

2. YOUR INNOVATION
   - What is your specific contribution?
   - How does it work?
   - What makes it different?

3. TECHNICAL DETAILS
   - Detailed technical description
   - Key components or steps
   - Diagrams or specifications
   - Mathematical formulas if applicable

4. ADVANTAGES
   - Performance benefits
   - Cost savings
   - Efficiency gains
   - Other advantages over existing solutions

5. IMPLEMENTATION
   - How would it be built/implemented?
   - What resources are needed?
   - What is the timeline?
   - What are the costs?

6. COMMERCIALIZATION
   - Who is the target market?
   - How large is the market?
   - What is commercial potential?
   - Who are your competitors?

Tips for strong disclosure:
- Be specific and detailed
- Use technical language appropriately
- Include diagrams and examples
- Be honest about limitations
- Emphasize novelty and value
- Make it easy for others to understand
```

## Troubleshooting for End Users

### Common Problems & Solutions

**Problem: "I forgot my password"**
```
Solution:
1. Go to login page
2. Click "Forgot Password?"
3. Enter your email
4. Check email for reset link
5. Click link and create new password
6. Try logging in with new password

If no email received:
- Check spam/junk folder
- Wait 5-10 minutes and check again
- Try resetting again
- Contact support if problem persists
```

**Problem: "File upload keeps failing"**
```
Solution:
1. Check file size (max 10MB)
2. Check file format (PDF, DOCX, etc.)
3. Try different file
4. Check internet connection
5. Close other browser tabs
6. Clear browser cache
7. Try different browser

If still failing:
- Make sure file is not corrupted
- Try uploading from different device
- Email support with file details
```

**Problem: "I can't see my evaluator's comments"**
```
Solution:
1. Refresh your browser (Ctrl+F5)
2. Go to My Submissions
3. Click your IP submission
4. Scroll down to "Evaluation Results"
5. Comments appear in gray box

If comments not showing:
- Wait 24 hours after evaluation complete
- Try different browser
- Log out and back in
- Contact supervisor for comments
```

**Problem: "My submission status hasn't changed in 2 weeks"**
```
Solution:
1. Check dashboard for status
2. Check email for any messages
3. Contact your supervisor:
   "Hello, my submission [ref#] 
    hasn't updated in 2 weeks. 
    Can you check the status?"

Typical timelines:
- Received: 2-3 days
- Under Review: 5-10 days
- Decision: 2-3 days
- Certificate: 1-2 days

If exceeding these times:
- Contact supervisor directly
- They can follow up with evaluators
```
